# Realtor Reports

Client-facing comparative market analysis reports, published via GitHub Pages.

Each report lives in its own folder (`<subject-slug>/index.html`) and is linked from the root [index.html](index.html) landing page.

## Adding a new report

1. Create a folder named after the property (e.g. `123-main-st/`).
2. Add the report as `123-main-st/index.html`.
3. Add a link to it in the root `index.html`.
4. Commit and push — GitHub Pages serves the `main` branch automatically.
